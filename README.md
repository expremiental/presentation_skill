# Presentation Skill

AI-скилл для создания интерактивных HTML-презентаций. Один файл, без зависимостей, открывается в браузере.

AI задаёт вопросы, собирает контент, показывает план — и только после OK собирает презентацию с анимациями, интерактивом и переходами.

## Как использовать

### Cursor

1. Скачай файл `cursorrules` из этого репозитория
2. Сохрани как `.cursorrules` в корне своего проекта
3. Открой чат в Cursor и напиши что хочешь сделать

Или одной командой в чате Cursor:

```
Скачай файлы cursorrules и example.html с https://github.com/expremiental/presentation_skill. Сохрани cursorrules как .cursorrules в корне текущего проекта, а example.html положи рядом.
```

### Claude Code

```bash
claude skill add --from https://github.com/expremiental/presentation_skill
```

## Как это работает

1. **Формат** — AI спрашивает про аудиторию, контекст показа, объём и вайб
2. **Контент** — AI задаёт конкретные вопросы, чтобы собрать материал на каждый слайд
3. **Саммари** — AI показывает план презентации, ждёт OK
4. **Сборка** — AI генерирует HTML-файл с анимациями и интерактивом

## Что получается

- Один `.html` файл — открывается в любом браузере
- Навигация стрелками, пробелом, кликером
- Анимации, интерактивные элементы (flip cards, таймлайны, счётчики)
- Адаптивный дизайн
